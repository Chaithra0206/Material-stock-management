# Material-stock-management

A complete Java-based material stock management project built using OOP and inheritance. 

Includes modules for:
- Users, Roles, and Permissions
- Products and Inventory Items
- Stock Tracking
- Customers and Billing
- Main class demonstrating full workflow

This project is structured in simple Java files (no packages), making it easy to run directly

## How to Run
1. Open the folder in VS Code.
2. Compile all `.java` files.
3. Run `Main.java`.

You will see sample outputs such as product listing, stock updates, and bill generation.
